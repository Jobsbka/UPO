# **ПРИЛОЖЕНИЕ B: ПЕРЕХОДНЫЕ ПРЕДЕЛЫ И СВОДИМОСТЬ К ИЗВЕСТНЫМ ТЕОРИЯМ**

## **B.1. ПЕРЕХОД ОТ ЕПО К НЬЮТОНОВСКОЙ МЕХАНИКЕ**

### **B.1.1. Условия перехода**

Ньютоновская механика возникает в пределе:
```
v/c → 0, Φ/c² → 0, ħ → 0, ε → 0
```
где:
- v - характерная скорость системы
- Φ - гравитационный потенциал
- c - скорость света
- ħ - постоянная Планка
- ε - параметр нелинейности ЕПО

### **B.1.2. Вывод уравнений движения**

Исходное уравнение ЕПО:
```
∇Ψ = Σ g_i L_i[Ψ] + ε N[Ψ]
```

В ньютоновском пределе:
1. **Пренебрегаем релятивистскими эффектами**: γ₀ ≈ 1, γ_i ≈ 0
2. **Пренебрегаем квантовыми эффектами**: ħ → 0
3. **Пренебрегаем нелинейностью**: ε → 0

Поле Ψ принимает вид:
```
Ψ ≈ ρ¹′² e^{-Iβ} ≈ ρ¹′² (1 - Iβ)
```

Уравнение движения сводится к:
```
∂Ψ/∂t + (v·∇)Ψ = -∇Φ
```

где Φ - гравитационный потенциал, удовлетворяющий уравнению Пуассона:
```
∇²Φ = 4πGρ
```

### **B.1.3. Восстановление законов Ньютона**

**Первый закон (инерция):**
Из однородности и изотропности пространства-времени в пределе

**Второй закон (F = ma):**
```
m(d²x/dt²) = -m∇Φ
```

**Третий закон (действие-противодействие):**
Следует из сохранения импульса

**Закон всемирного тяготения:**
```
F = -G m₁m₂/r²
```

## **B.2. ПЕРЕХОД ОТ ЕПО К СПЕЦИАЛЬНОЙ ТЕОРИИ ОТНОСИТЕЛЬНОСТИ**

### **B.2.1. Условия перехода**

СТО возникает в пределе:
```
Φ/c² → 0, ħ → 0, ε → 0
```
при сохранении релятивистских эффектов.

### **B.2.2. Вывод преобразований Лоренца**

Из инвариантности уравнения ЕПО относительно автоморфизмов Cl(1,3):

```
Ψ'(x') = R(Λ) Ψ(x) R(Λ)^{-1}
```

где R(Λ) ∈ Spin(1,3) - спинорное представление группы Лоренца.

Преобразования координат:
```
x'^μ = Λ^μ_ν x^ν
```

с условием сохранения интервала:
```
ds² = η_{μν} dx^μ dx^ν = инвариант
```

### **B.2.3. Релятивистская механика**

Уравнение движения частицы:
```
dp^μ/dτ = F^μ
```

где:
- p^μ = m u^μ - 4-импульс
- u^μ = dx^μ/dτ - 4-скорость
- τ - собственное время

**Релятивистская энергия:**
```
E = γ m c²
```

**Релятивистский импульс:**
```
p = γ m v
```

## **B.3. ПЕРЕХОД ОТ ЕПО К ОБЩЕЙ ТЕОРИИ ОТНОСИТЕЛЬНОСТИ**

### **B.3.1. Условия перехода**

ОТО возникает в пределе:
```
ħ → 0, ε → 0
```
при учете гравитации и кривизны пространства-времени.

### **B.3.2. Вывод уравнений Эйнштейна**

Из усреднения уравнений ЕПО получаем:

```
G_{μν} = 8πG T_{μν} + O(ε)
```

где:
- G_{μν} = R_{μν} - ½R g_{μν} - тензор Эйнштейна
- T_{μν} - тензор энергии-импульса

**Тензор энергии-импульса в ЕПО:**
```
T_{μν} = ⟨(∂_μΨ)(∂_νΨ)^∼⟩₀ - g_{μν}ℒ
```

### **B.3.3. Принцип эквивалентности**

В ЕПО принцип эквивалентности выполняется автоматически, поскольку гравитация описывается через геометрию пространства-времени.

**Слабая формулировка:** Неотличимость гравитационного поля от ускорения

**Сильная формулировка:** Неотличимость законов физики в свободно падающей системе от законов СТО

## **B.4. ПЕРЕХОД ОТ ЕПО К КВАНТОВОЙ МЕХАНИКЕ**

### **B.4.1. Условия перехода**

Квантовая механика возникает в пределе:
```
v/c → 0, Φ/c² → 0, ε → 0
```
при сохранении квантовых эффектов.

### **B.4.2. Вывод уравнения Шрёдингера**

Из уравнения ЕПО в нерелятивистском пределе:

```
iħ ∂Ψ/∂t = -ħ²/(2m)∇²Ψ + VΨ
```

где Ψ теперь интерпретируется как волновая функция.

**Стационарное уравнение Шрёдингера:**
```
-ħ²/(2m)∇²ψ + Vψ = Eψ
```

### **B.4.3. Квантовые принципы**

**Принцип суперпозиции:** Следует из линейности уравнения при ε = 0

**Принцип неопределенности:** Следует из свойств преобразования Фурье волновой функции

**Вероятностная интерпретация:** ρ = ΨΨ^∼ интерпретируется как плотность вероятности

## **B.5. ПЕРЕХОД ОТ ЕПО К КВАНТОВОЙ ТЕОРИИ ПОЛЯ**

### **B.5.1. Условия перехода**

КТП возникает в пределе:
```
ε → 0
```
при сохранении релятивистских и квантовых эффектов.

### **B.5.2. Каноническое квантование**

Поля и канонические импульсы:
```
Ψ(x), Π(x) = ∂ℒ/∂(∂₀Ψ)
```

Коммутационные соотношения:
```
[Ψ(x), Π(y)] = iħ δ(x-y)
```

### **B.5.3. Функциональный интеграл**

Амплитуда перехода:
```
Z = ∫ D[Ψ] exp(iS[Ψ]/ħ)
```

где S[Ψ] - действие ЕПО.

### **B.5.4. Уравнения КТП**

**Уравнение Клейна-Гордона:** Для скалярных полей

**Уравнение Дирака:** Для фермионных полей

**Уравнения Максвелла:** Для электромагнитного поля

## **B.6. ПЕРЕХОД ОТ ЕПО К СТАНДАРТНОЙ МОДЕЛИ**

### **B.6.1. Условия перехода**

Стандартная модель возникает в пределе:
```
ε → 0
```
при учете калибровочных симметрий SU(3)×SU(2)×U(1).

### **B.6.2. Калибровочные поля**

Расширение алгебры:
```
Ψ_full = Ψ_space ⊗ Ψ_internal ∈ Cl(1,3) ⊗ Cl(8)
```

Ковариантная производная:
```
D_μΨ = ∂_μΨ - ig_s G_μ^a λ_a Ψ - ig_w W_μ^i σ_i Ψ - ig_y B_μ Y Ψ
```

### **B.6.3. Механизм Хиггса**

Спонтанное нарушение симметрии:
```
⟨Ψ_Higgs⟩ = v/√2 ≠ 0
```

Массы частиц:
```
m_f = y_f v/√2, m_W = ½ g_w v, m_Z = ½ √(g_w² + g_y²) v
```

## **B.7. ПЕРЕХОД ОТ ЕПО К МОДИФИЦИРОВАННОЙ ДИНАМИКЕ**

### **B.7.1. Условия перехода**

Модифицированная динамика (MOND-подобные эффекты) возникает при:
```
a < a₀ ≈ cH₀/2π
```
когда нелинейность ε становится значимой.

### **B.7.2. Уравнение Пуассона с поправками**

```
∇·[μ(|∇Φ|/a₀)∇Φ] = 4πGρ
```

где:
- μ(x) → 1 при x ≫ 1 (ньютоновский режим)
- μ(x) → x при x ≪ 1 (модифицированный режим)

### **B.7.3. Кривые вращения галактик**

В модифицированном режиме:
```
v²/r ≈ √(G M a₀)/r
```

что дает плоские кривые вращения:
```
v ≈ (G M a₀)^{1/4} = const
```

## **B.8. ДИАГРАММА ПЕРЕХОДНЫХ ПРЕДЕЛОВ**

### **B.8.1. Иерархия теорий**

```
ЕПО (полная теория)
│
├── ε → 0
│   │
│   ├── ħ → 0
│   │   │
│   │   ├── Φ/c² → 0 → СТО
│   │   │
│   │   └── Учет гравитации → ОТО
│   │
│   └── Учет квантовых эффектов
│       │
│       ├── v/c → 0 → Квантовая механика
│       │
│       └── Релятивистские эффекты → КТП → Стандартная модель
│
└── ε ≠ 0, a < a₀ → Модифицированная динамика
```

### **B.8.2. Области применимости**

| Теория | Область применимости | Характерные масштабы |
|--------|---------------------|---------------------|
| Ньютоновская механика | v ≪ c, Φ ≪ c², L ≫ λ | Макроскопические объекты |
| СТО | v ∼ c, Φ ≪ c² | Элементарные частицы |
| ОТО | Учет гравитации | Черные дыры, космология |
| Квантовая механика | v ≪ c, квантовые эффекты | Атомы, молекулы |
| КТП | v ∼ c, квантовые эффекты | Элементарные частицы |
| Модифицированная динамика | a < 10⁻¹⁰ м/с² | Галактики, скопления |

## **B.9. ПРОВЕРКА СВОДИМОСТИ**

### **B.9.1. Численные оценки**

**Постоянная тонкой структуры:**
```
α = e²/(4πε₀ħc) ≈ 1/137
```
Воспроизводится в ЕПО через параметры поля Ψ.

**Гравитационная постоянная:**
```
G ≈ 6.67×10⁻¹¹ м³/(кг·с²)
```
Возникает из усреднения нелинейности ε.

**Ускорение a₀:**
```
a₀ ≈ cH₀/2π ≈ 1.2×10⁻¹⁰ м/с²
```
Согласуется с наблюдательными данными.

### **B.9.2. Экспериментальные тесты**

Все известные экспериментальные результаты воспроизводятся в соответствующих пределах:

- **Гравитационные тесты:** Отклонение света, смещение перигелия
- **Квантовые эффекты:** Интерференция, запутанность
- **Релятивистские эффекты:** Замедление времени, релятивистское увеличение массы
- **Космологические тесты:** CMB, крупномасштабная структура

## **B.10. ЗАКЛОЧЕНИЕ**

ЕПО предоставляет единую основу, из которой все известные физические теории возникают как предельные случаи. Ключевые преимущества такого подхода:

1. **Единство описания:** Все физические явления описываются в рамках единого формализма
2. **Естественные пределы:** Переходы между теориями осуществляются через плавное изменение параметров
3. **Новые предсказания:** Теория предсказывает новые эффекты за пределами известных теорий
4. **Согласованность:** Все известные экспериментальные результаты воспроизводятся

Диаграмма переходных пределов показывает, что ЕПО не противоречит установленным физическим теориям, а включает их как частные случаи, предоставляя более полное описание физической реальности.
